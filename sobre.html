Sobre o Projeto

Este projeto é uma galeria de imagens interativa, desenvolvida para demonstrar o consumo de APIs públicas e manipulação de dados no frontend.

Funcionalidades principais:

  • Visualização de imagens de raças de cães usando a API Dog CEO.
  • Cadastro e listagem de usuários simulados com a API Reqres.in.
  • Interface responsiva e amigável.
O objetivo é praticar integração com APIs.
